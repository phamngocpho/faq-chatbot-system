# FAQ Admin Frontend

Admin panel để quản lý câu hỏi FAQ.

## Cài đặt

```bash
npm install
```

## Chạy development server

```bash
npm run dev
```

Mở trình duyệt tại: http://localhost:5173

## Build production

```bash
npm run build
```

## Lưu ý

- Đảm bảo Backend API đang chạy ở http://localhost:3000
- Nếu Backend chạy ở port khác, sửa `API_URL` trong file `src/App.vue`
